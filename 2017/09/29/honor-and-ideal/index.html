荣誉和理想

发表于 2017-09-29
字数统计 1,119 字

今天有两件事值得说说。

第一件事，跟一本杂志有关。准确的说，是关于这本杂志的专题内容——《瘦瘦瘦，笔记本减肥真的那么容易？》。作者以时间为序，以“减肥”为主线，详细介绍了笔记本形态的工业发展历程。激动的我，几乎是一字一字数着看完全篇的。现在我只能用“颠覆”一词去形容一路走来的笔记本电脑。

作为计算机时代的重要产物之一，笔记本进化三十余载，大浪淘沙，结出了丰硕的果实。标志性如1992年诞生的IBM ThinkPad 700C，第一次告诉世界“ 笔记本电脑 ”应该长什么样子。此后，索尼 VGN-UX17C，华硕 EeePC 700，以及各式各样的 Atom 平板，相继出炉。2008年，东芝 Portege R502，12.1英寸，体重仅有 900g。2009年，戴尔超薄 Adamo XPS 发布，机身厚度 9.99mm，纪录一直保持到2016年才被宏碁 Swift 7 打破（9.98mm）。2011年底，Ultrabook 超极本概念的诞生，标志着笔记本形态的发展进入了新的里程碑。Intel 携一众移动超低电压 Core™ CPU 登场，使得笔记本“ 小巧 + 高性能 + 省电 ”成为现实。后期，随着散热器的设计和材料的不断改良，笔记本厚度再次缩小。相信 2008 年乔布斯从一个信封里掏出 MacBook Air 时，没有人不由衷赞美苹果的设计，但是很少有人知道，MacBook 所使用的黑科技经历了怎样的变革。

这些东西带给我什么呢？美。科技之美，工匠之美，工程师之美。我闭上眼睛，感受着，自己有一双智慧的手，正在设计草图里驰骋，在机床上精雕细琢、打磨、成型……成就感油然而生，没有什么比自己的作品流传于世更让人激动不已的了。这种荣誉，属于真正的创造者——工程师，且不管他们有没有共鸣，我反正已经陶醉得不能自拔了。

第二件事，关于某某的同学拿到澳洲 PR 的事情。我想，每一个到过澳洲的 WHVer，大抵都被人问过“ 想不想移民 ”的问题。我甚至有一叠律师楼经手的客户签证档案。

老实说，我想移民，如果可以。移民的好处很多，人的一生可能因此被改写，也可能依然平凡，不好说。但是，在看得见的资本主义社会里，你有权自由自在地上“谷歌”、批判时政而不必因言获罪，你有结社自由、宗教信仰自由、新闻自由、人之免于恐惧的自由等等……但是，我深知，自由需要付出代价，你必须拿点东西来交换，有可能是买路钱，也有可能是鲜血。移民？呵，没有破釜沉舟的意志支撑，穷人的移民路注定是万里荆棘。

人说，移民是一项大工程，所言不虚。如果你是富翁，那么好办，几千万可以给全家人买资本主义的入场券。而如果你像我一样，一介平民，那么除了留学，婚嫁，雇主担保，你不再有其他选择。留学，先投入后产出，没有实打实的钞票和才高八斗的智慧，此路不通。婚嫁，高富帅和白富美成功率高，普通人别白日梦了，况且还有文化差异门槛在。雇主担保，不难不易，命运掌握在雇主手里，时间会告诉你，这是先苦后苦的招式……

其实，我个人挺害怕失败的，不敢放手一搏。担心自己的智商达不到谈论努力的程度。荷包干瘪，挤不出钞票，唯有挤出时间卖劳力。英文和情商，双弱鸡水平，只能生存，无法谈情说爱……但是怎么办呢，我并不打算放弃自己，我想要看见，即使不能移民，也要做最好的自己。

命里有时终须有，命里无时莫强求。我佩服走钢丝的人，成也罢，败也罢，我认为都是一条好汉。勇敢去做，已经赢了一半，剩下一半，交给老天爷。没有成功，也会有成长。

总有一天，自由会站在你面前，迎着风，向你招手。

夜里北京
